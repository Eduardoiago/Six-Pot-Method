Six Pot


The Six Pot project was developed using HTML, CSS and JavaScript. We used the JS libraries: Chart to convert the calculation into a graph, and Toastify to alert us of invalid functions. This financial organization strategy can be applied with different objectives: keeping bills paid, building up an emergency reserve, paying off debts, making a retirement plan, achieving goals, etc.


Version 1.0 of the project was created just to analyze the functioning and layout of the tool.


Default Setting:


  • 40% for Working capital.
  • 20% for Retirement.
  • 15% for Emergency box.
  • 10% for Investments.
  • 10% for Entertainment.
  • 05% for Donations and Gifts.
